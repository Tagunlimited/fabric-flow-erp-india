To deploy the reset-employee-password edge function:

OPTION 1: Via Supabase Dashboard (Easiest)
1. Go to: https://supabase.com/dashboard/project/vwpseddaghxktpjtriaj/functions
2. Click "Create a new function"
3. Name: reset-employee-password
4. Copy the code from: supabase/functions/reset-employee-password/index.ts
5. Click Deploy

OPTION 2: Via CLI
supabase functions deploy reset-employee-password

After deployment, the password reset will work!
